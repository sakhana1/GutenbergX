**GutenbergX**

GutenbergX is a concept UI product that transforms free classic books from Project Gutenberg into a modern, cinematic reading experience because, that contains legacy. The Main goal of this project is to make reading classic literature more engaging, interactive, and visually appealing.

**Features**

Clean and modern UI design for classic books.

Easy navigation through chapters.

Responsive layout for desktop and mobile.

Focus on readability and user experience.

**My Contribution**

I worked on designing and developing the frontend of the website, ensuring:

Proper layout and typography.

Smooth navigation between pages.

Visual aesthetics that enhance reading.

**Tech Stack**

HTML, CSS, JavaScript
